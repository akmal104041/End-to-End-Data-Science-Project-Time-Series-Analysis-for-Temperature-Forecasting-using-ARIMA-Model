

## Requirements

* Python 3.5 or higher
* Numpy
* Pandas
* Matplotlib
* Statsmodels

## Dataset

In this project, the Dataset used is the famous Kaggle [Dataset](https://www.kaggle.com/berkeleyearth/climate-change-earth-surface-temperature-data).
